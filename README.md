📘 TOEFL iBT 2026: Reading Practice Simulator
This is a web-based interactive tool designed to simulate the "Complete the Word" reading tasks for the TOEFL iBT. It helps students improve their academic vocabulary, context recognition, and typing speed under timed conditions.

🚀 Key Features
Timed Sessions: A global 5-minute countdown to simulate exam pressure.

Automatic Word Splitting: The script dynamically hides the second half of academic words, requiring the user to complete them.

Instant Feedback: "Check Answers" provides color-coded results (Green for correct, Red for incorrect).

Academic Variety: Includes passages from Biology, Art History, Psychology, and Environmental Science.

Progress Tracking: Tracks your score across 5 different passages and provides a final accuracy percentage.

🛠️ Technical Setup
HTML5/CSS3: Used for the clean, exam-like interface.

Vanilla JavaScript: Handles the timer logic, string manipulation for word gaps, and scoring.

Responsive Design: Works on both desktop and mobile browsers.

📖 How to Use
Click the Start Timer button at the bottom to begin.

Read the first sentence (provided for context).

Fill in the missing letters in the subsequent sentences.

Click Check Answers to see your score for that passage.

Click Next Passage to continue until the summary screen appears.
